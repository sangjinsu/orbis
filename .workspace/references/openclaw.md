# Reference: OpenClaw

## Why This Matters

OpenClaw concepts remain useful comparison points, but Orbis is not an
OpenClaw clone. The shipped v2.1 baseline already includes ordered session
lanes, asynchronous runtime events, named-role mutation auth, and a live global
skill lifecycle feed.

## Concepts Already Reflected

- session lane ordering and immediate ACK plus asynchronous progress events
- cancellation and timeout ownership in the runtime
- named reviewer/admin auth for mutations
- live global skill lifecycle observation

## Concepts to Evaluate Only Through a Future Accepted Milestone

- steering and follow-up queues
- context reports
- broader gateway pairing
- global concurrency caps

## Concepts Not Currently Adopted

- OpenClaw compatibility
- session tools or multi-agent session visibility
- multi-channel gateway adapters
